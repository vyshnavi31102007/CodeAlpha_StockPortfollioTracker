📈 Stock Portfolio Tracker

📌 Project Description

Stock Portfolio Tracker is a simple Python project that helps users calculate the total investment value of their stock portfolio.

Users can:

- Enter the number of stocks they own
- Input stock names and quantities
- Calculate total investment automatically
- Save results into a text file ("portfolio.txt")

This project uses hardcoded stock prices for calculation.

---

🚀 Features

✔ Track multiple stocks
✔ Calculate investment value automatically
✔ Supports predefined stock symbols
✔ Save portfolio results into a text file

---

🛠 Technologies Used

- Python
- File Handling
- Dictionary
- Loops
- Conditional Statements

---

▶ How to Run

1. Install Python
2. Copy the code into a Python file ("stock_tracker.py")
3. Run the program:

python stock_tracker.py

4. Enter stock details when prompted.
